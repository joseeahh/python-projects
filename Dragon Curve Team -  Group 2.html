Now, let's construct a function that will plot the dragon curve at any iteration, with **kwargs

In [6]:
def dcurve(n, cw, Y, c1 = 'blue', **kwargs):
    '''
    This function will accept 4 arguments, and return the nth iteration of the dragon curve. cw- denotes how much
    the next iteration will rotate. Y is an array that we clarify beforehand, and **kwargs is a wrapper intended allow
    the user to change the color of the rotation
    '''
    X = np.array ([
        [0,-1*cw],
        [cw,0]
    ])
    for i in range(1,n):
        Rotation = Y[::-1] @ X
        Y = np.concatenate([Y,Rotation[1:]])
        Y = Y - Y[len(Y) - 1]
        L = len(Y)
        Y1 = Y[:L//2 + 1]
        Y2 = Y[L - (L//2 + 1):]
    plt.plot(Y1[:,0], Y1[:, 1], c1)
    plt.plot(Y2[:,0], Y2[:, 1], **kwargs)
    plt.axis('equal')
    #print(Y, len(Y))
    plt.title('This is the dragon curve with ' +str(n) + ' iterations' )

So if the function is working properly, then the iteration should return 2 line segments, similar to what we have above if we input 2 as the argument for n, and -1 as the argument for cw
